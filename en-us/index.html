          # EYins

          As you can see, Hexo has added another theme for Labor Day 2020 - EYins.

          EYins -- simplicity is not simplicity.

          Performance, beauty, SEO, everything.

          # Summary

          This is an age of information explosion.

          Blogging, being able to record our bits and pieces and share them with others.

          "Hexo" is an open source blogging framework based on Node.js.

          At hexo.io/themes, there are many excellent topics.

          Of course, there is also a part of the theme that dazzles us, we love flat winds and clean design, and to satisfy that desire, this theme appears.

          EYins - EasyYins, simple configuration, powerful features, as said above: simple, not simple.

          # Acknowledgements

          Due to my lack of skills, I wanted to find a theme template based on the overall revision,

          I wanted to use Hexo's default theme - "landspace", but I don't know much about stylus...

          So on Labor Day, I found the theme for wa-ri - ztopic, this theme wa-ri cooed for a long time, so that day I applied to him in Issues for the right to second production, he agreed.

          This theme is officially in the works, and here's a special thanks to WA-RI for agreeing to it!!!!

          # Advantages

          On the basis of ztopic, I have added a part of the content, especially in the missing part for LightHouse optimization, currently my blog (using EYins) LightHouse score as follows:

          Not only the score, but also the interface of the theme is very nice!

          Theme DEMO

          Adaptive is also perfect, version 0.15 theme size (with background image removed):103KB

          # Powerful

          No need to touch complex code, just customize it all in the configuration file.

          Support for 3 common commenting systems (to be added gradually): Disqus, Valine, Birelli

          Supports 3 statistical services: Google Analytics, Baidu Statistics, Tencent Mobile Analytics

          The background, footer, and copyright can all be adjusted to create your own website style.

          If you have any questions about the use of the site, please feel free to ask them at Issues

          But... Suggest checking/checking the Q&A yourself before asking questions.

            # Configuration items

            Please use editors such as Notepad++, VSCode, etc. to open the theme profile. For Windows users, it is not recommended to open the configuration file with your own Notepad, which is inconvenient and can cause unpredictable coding errors.

            # Other

            favicon:Image path

            favicon: /favicon.ico

            Description:Set the avatar in the title of your blog page (support extranet)

            # Plug-in Management

            fancybox:Is it on?

            fancybox: true

            Description:Set if you want to open fancybox

            # Navigation bar

            Navigation bar content:Guide path

            menu:
  home: /archives
  archive: /archives
  category: /categories
  tags: /tags
  about: /about
  RSS: /atom.xml

            Description:Set the navigation bar content and the path of the navigation bar.

            # Website logo and footer

            logo:your nickname

            logo: LiangYin

            Description:Show the nickname on the left side of the navigation bar at the top of the blog (mobile side is shown in the middle)

            banner:Image path

            banner: img/background.jpg

            Description:Display the background image at the top of the PC blog page, support the extranet (mobile phone does not display)

            quote:string

            quote: "The long wind and waves will break sometimes, and the straight clouds and sails will help the sea. --"Li Bai"

            Description: Sentences displayed at the bottom of the blog page

            # Post_Copyright

            copyright:copyright information

            copyright: "<p style='padding:none'>This article by LiangYin</p><p style='padding:none'>is strictly forbidden to be reprinted without the author's permission unless otherwise indicated! </p>"

            Description:Display the copyright information at the bottom of the article, if empty, it will not be displayed (support HTML syntax)

            end:string

            ``end: the end`''

            Description:Ending logo displayed under copyright information

            # Social networks

            Social Media Name:Links

            SOCIAL:
  zhihu: 
  weibo: 
  stackflow:
  github:

            Description:The social media information displayed at the bottom of the page is only supported for the above.

            # Comment system.

            # valine comment system

            valine:
  comment_name: comment # your comment area title comment title
  enable: false # if you want use valine, please set this value to true enable valine
  app_id: # your leancloud appId
  app_key: # your leancloud appKey
  # Comment Box placeholder
  pageSize: 8 # Number of pages displayed for comments
  enableQQ: true # Whether to convert QQ number to QQ nickname and avatar
  # Custom emoji feature not yet developed in current thread, Big Brother is welcome to submit PR.
  # emojiCDN: ''
  # emojiMaps:

            Please go to Valine's official website for how to use it

            # Third-party statistics/inclusion services

              baidu:  # Your APPID
  tencent: # Your APPID
  google: # Your ID
  baidu_sitemap: true # Not "empty" means active.

            For the time being, only the above service providers are supported, Baidu site map submission is not enabled as "empty".

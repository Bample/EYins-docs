            # Q & A

            # 为什么我的博客页面默认展示的是全部的博文内容？

            请在文章中使用<!-- more -->手动进行截断

            # 为什么$ hexo server显示"No layout: index.html"？

            检查是否正确下载了主题,或检查是否在站点配置文件中切换了主题。

            # 无法加载样式表？

            是否安装了sass $ npm install --save hexo-renderer-sass

            # 为何新增的菜单项显示 Menu.xxx？

            参见:NexT使用文档

            # 为何valine的自定义表情功能无法使用？

            此功能正在开发中，欢迎提交pr，补充。

          # 摘要

          這是一個信息爆炸的時代。

          博客，能記錄下我們的的點點滴滴，並與他人分享。

          「Hexo」是一個基於Node.js的開源博客框架。

          在hexo.io/themes，有許多優秀的主題。

          當然，也有一部分使我們眼花繚亂的主題，我們喜歡扁平風與簡潔的設計，為了滿足這個慾望，這款主題出現了。

          EYins - EasyYins，簡單的配置，強大的功能，正如上面所說: 簡潔 ，不簡單。

          # 鳴謝

          由於自己技術的欠缺,想找一個主題模板的基礎上進行整體的修改,

          本來想使用Hexo的默認主題 - 「landspace」但是自己不是很懂stylus...

          於是在勞動節這天，我找到了wa-ri的主題- ztopic,這款主題wa-ri咕咕咕了很久,於是這天我在Issues向他申請二次製作的權利，他同意了。

          這款主題正式開工，在這里特別謝謝wa-ri的同意！ ！ ！

          # 強大

          無需接觸複雜的代碼，只要在配置文件中就可以通通自定義。

          支持 3 種通用評論系統(之後會逐步增加)：Disqus、Valine、必來力

          支持 3 種統計服務：Google Analytics、百度統計、騰訊移動分析

          背景，頁腳，版權 均可隨心調整，打造專屬自己網站的樣式。

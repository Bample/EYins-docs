            當前文檔描述的是主題的0.15版本

            在本則文檔中，認定您已經會使用Hexo

            # 配置項

            請使用 Notepad++，VSCode等編輯器打開主題配置文件。對於 Windows 用戶，不建議使用自帶記事本打開配置文件，既不方便,也會帶來不可預料的編碼錯誤。

            # 其它

            favicon:圖片路徑

            favicon: /favicon.ico

            說明:設置您博客頁面標題處的頭像(支持外鏈)

            # 插件管理

            fancybox:是否開啟

            fancybox: true

            說明:設置您是否開啟fancybox

            # 導航欄

            導航欄內容:導向路徑

            menu:
  home: /
  archive: /archives
  category: /categories
  tags: /tags
  about: /about
  RSS: /atom.xml

            說明:設置導航欄內容與導航欄導向的路徑

            # 網站標誌和頁腳

            logo:你的暱稱

            logo: LiangYin

            說明:顯示在博客頂部導航欄左側的暱稱(手機端顯示在中部)

            banner:圖片路徑

            banner: img/background.jpg

            說明:顯示在PC端博客頁面頂部的背景圖片,支持外鏈(手機端不顯示)

            quote:字符串

            quote: "長風破浪會有時，直掛雲帆濟滄海。——李白"

            說明:顯示在博客頁面底部的句子

            # Post_Copyright

            copyright:版權信息

            copyright: "<p style='padding:none'>本文作者 LiangYin</p><p style='padding:none'>除非特殊標明,未經作者允許嚴禁轉載!</p>"

            說明:顯示在文章底部的版權信息,為空則不顯示(支持HTML語法)

            end:字符串

            end: the end

            說明:顯示在版權信息下的結尾標識

            # 社交網絡

            社交媒體名稱:鏈接

            social:
  zhihu:
  weibo:
  stackflow:
  github:

            說明:顯示在頁面底部的社交媒體信息,現僅支持上述內容。

            # 第三方服務設置

            # 評論系統

            # valine評論系統

            valine:
  comment_name: 評論 # your comment area title 評論標題
  enable: false # if you want use valine, please set this value to true 啟用valine
  app_id: # your leancloud appId
  app_key: # your leancloud appKey
  placeholder: 來留言呀~ # Comment Box placeholder 佔位符
  pageSize: 8 # 評論顯示頁數
  enableQQ: true # 是否把qq號轉換為qq暱稱和頭像
  # 當前主題還未開發自定義emoji功能，歡迎大佬提交pr。
  # emojiCDN: ''
  # emojiMaps:

            請到Valine官方網站獲取如何使用它

            # 自定義emoji示例

            # disqus評論系統

            disqus_shortname:

            請在disqus官網獲取你的shortname

            # 必來力評論系統

            livere_uid:

            將 livere_uid 改為自己的uid則啟用該評論系統

            # 第三方統計/收錄服務

              baidu: # Your APPID
  tencent: # Your APPID
  google: # Your ID
  baidu_sitemap: true # Not "empty" means active.

            暫時僅支持上述服務商,百度站點地圖提交不啟用為"空"即可。

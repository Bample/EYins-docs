            # Q & A

            # 為什麼我的博客頁面默認展示的是全部的博文內容？

            請在文章中使用<!-- more -->手動進行截斷

            # 為什麼$ hexo server顯示"No layout: index.html"？

            檢查是否正確下載了主題,或檢查是否在站點配置文件中切換了主題。

            # 無法加載樣式表？

            是否安裝了sass $ npm install --save hexo-renderer-sass

            # 為何新增的菜單項顯示 Menu.xxx？

            參見:NexT使用文檔

            # 為何valine的自定義表情功能無法使用？

            此功能正在開發中，歡迎提交pr，補充。
